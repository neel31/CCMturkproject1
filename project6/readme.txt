The framework of this game is based off the one found on https://github.com/irealva and was discussed with Isha Mehra. 
(cited code in index.html and app.js.)
Images are cited from the following URLs:
http://noelnegusse.com/img/projects/space_invader2.png
http://www.softicons.com/game-icons/classic-games-icons-by-thvg/space-invaders-1-icon


The purpose of this game is to have multiple parties playing a game, in the style of Twitch Plays Pokemon.
Multiple players attempt to survive the obstacles, but the position of the piece is based off the collective median location of all the players of the game.
